# Телеграмм бот для взаимодействия с ChatGPT

## Установка
- Установить poetry
### Команды

```bash
poetry install
python -m app
```
## Использование
### Команды
- start - сброс к началу
- add_role - команда для ввода роли ChatGPT для диалога
- clear - очистка диалога
- image - создать изображение
- settings - настройки бота
## Технологии
- Python 3.11
- SQLite
- Aiogram
- SQLAlchemy
- Alembic
